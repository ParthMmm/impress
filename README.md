# Impress

Share and find mechanical keyboard switches.

## Features
- User registration
- Upload images & posts 
- Liking & Disliking
- Dark mode compatibility
- Responsive design

## Built With
- React
- Redux
- MongoDB
- Express
- Node.js
- TailwindCSS
- React-Router
- Password
- JWT
